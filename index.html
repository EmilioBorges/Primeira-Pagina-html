Surgimento da Internet e do HTML

(inicio)

No início os principais meio de comunicação era atraves do uso de telégrafos e o telefone, os computadores eram grandes maquinas utilizadas para grandes calculos e armazenar informações ou seja seu principal uso era para fins científicos e governamentais. Em 1957 durante a Guerra fria protagonizada por Estados Unidos e União Soviética, foi iniciado variados estudos com o objetivo de encontrar soluções para melhor proteger as informações e as comunicações. Em 1958 foi criada a DARPA nos Estados Unidos uma agência de pesquisas avançadas, em 1961 Leonard Kleinrock apresentou com sucesso sua teoria onde foi provado que dois servidores poderiam se comunicar para enviar e receber informações por pacotes por meio de uma rede de nós. Em 1989 Tim Berners-Lee desenvolveu a World Wide Web que funciona como um ssitem de distribuição de decumentos de hipertxto(HTTP) interconectados e acessíveis por meio de um navegador web conectado a internet. O sistema ficou tão popular que em 1991 foi aberto ao público, isso foi possível graças à criação do navegador Mosaic em 1993.

Como funciona o HTM

(inicio)

HTML(HyperTExt Markup Language) é uma linguagem de marcação de texto que permite apresentar informações na internet, como links, imagens, textos e tambem audio. Basicamente um site e feito com uso de tags que possuem seus atributos, temos as tags principais que sao a HTMl que cria a base do site, depois temos a head que e o cabeçalho do site, body definido como o corpo do site, temos a tag img destinada para uso de imagen ou audio para uso de audios.

O que sao Tags no HTML

(inicio)

Tags são marcadores que são responsaveis de comunicar ao browser como deve ser lido cada arquivo de acordo com cada tag e seus atributos que foram usados no desenvolvimento, mostrando de forma correta como a pagina foi desenvolvida. Cada tag tem suas caracteristicas, alguma tem tag de abertura e fechamento, e temos tambem as que nao e necessario a tag de fechamento, por exemplo temos h1 que uma tag de textos onde seu uso e feito h1 e usando /h1 para indicar seu fechamento, diferentemente da tag img que nao tem a taga de fechamendo podendo ser usado a / ao final apenas como indicação do fechamento

Tipos de atributos no HTML

(inicio)

Atributos são informações adicionais que podem ser adicionadas dentro das tags que podem ser do tipo id que podem ser usados uma unica vez, ou class que podem ser usadas mais de uma vez, esses atributos sao usados no Javascript ou no css, atributos podem ser utilizados para mostrar o caminho de uma imagem que esta dentro da pasta do projeto ou uma url dentre outros.

Uso de imagens com HTML

(inicio)

Para utilizarmos um imagem dentro de um site, e criado uma pasta imagem dentro do projeto, onde e utilizado a tag img mostrando ao browser que temos uma imagem ali, e utilizando seus atributos e possivel designar o caminho onde esta imagem esta salva dentro do projeto ou ate mesmo de outro site usando a url, sendo possivel tambem configurar o tamanho da imagem usando os atributos

Tags para Texto no HMTL

(inicio)

Para o uso de texto temos algumas tags definidas para tals fins, uma delas seria a tag p que sigifica paragráfo, temos a tag h que vai da h1 ate h6, esta tag e utilizada para o uso de titulos, quando menor o numero maior será o tamanho da fonte. Temos tambem as tag que sao especificas para o uso de listas que seri a 0l(ordenada) e a ul(não ordenada), onde uma e feita de forma ordenada e a outra e feita de forma desordenada e a li que vai representar cada elemento da lista.
